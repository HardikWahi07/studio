# **App Name**: TripMind

## Core Features:

- AI Itinerary Generation: Generate personalized itineraries based on user preferences (destination, budget, interests) using AI, offering eco-friendly options. The LLM can use a tool to assess which options will reduce carbon impact.
- Smart Transport Recommendation: Compare travel modes (train, bus, flight) based on cost, time, and carbon footprint to suggest sustainable routes. Uses external APIs.
- Hidden Gems Explorer: Discover lesser-known authentic experiences, includes a 'Spontaneity Button' for random exploration, using a generative AI tool for suggestions.
- Local Business Connector: Connect travelers with local artisans and experiences to promote community-based tourism, linking to local services via APIs.
- Group Expense Splitter: Log shared trip expenses and automatically calculate who owes whom, simplifying group travel finances.
- Location & Safety Companion: Provide real-time group location, safety alerts, and access to nearby hospitals. Integrates with a safety chatbot using AI.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to reflect sustainability and nature.
- Background color: Light beige (#F5F5DC), for a clean and calming feel.
- Accent color: Earthy brown (#A1887F) for highlighting local business connections.
- Headline font: 'Playfair' (serif) for an elegant and fashionable feel; body text: 'PT Sans' (sans-serif) for readability.
- Use eco-friendly and travel-themed icons (e.g., leaves for sustainability, map pins for location).
- Clean, card-based layout to showcase destinations, itineraries, and local experiences clearly.
- Subtle transitions and loading animations to enhance user experience without being distracting.
